# Port Usage Monitor

A lightweight tool for **monitoring and limiting port traffic** on Linux servers.  
You can set **data limits (MB)** and **time limits (minutes)** for each port.  
When a limit is reached, the port will automatically be **blocked**.

Perfect for Tunneling
